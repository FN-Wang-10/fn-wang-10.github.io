过度和动画的区别

相同点

都是随着事件改变属性值

不同点

  1. 过度需要用户主动触发，动画不需要
  2. 过度需要中间值
  3. 过渡只执行一此，动画是一直执行
  4. 动画可以完成更精准的操作
